# 病虫害识别系统架构说明

## 1. 系统架构

```
病虫害识别系统
├── 前端界面层（PyQt5）
│   ├── 图像采集模块
│   ├── 用户交互界面
│   └── 结果展示模块
│
├── 后端处理层
│   ├── 图像预处理模块
│   ├── 模型推理模块
│   └── 数据存储模块
│
└── 模型层（YOLOv8）
    ├── 模型加载
    ├── 推理加速
    └── 结果输出
```

## 2. 技术栈

### 2.1 开发语言
- Python 3.x
- 主要依赖库：
  - PyQt5：GUI界面开发
  - OpenCV：图像处理
  - PyTorch：深度学习框架
  - Ultralytics YOLO：目标检测模型

### 2.2 开发工具
- PyCharm：IDE
- Qt Designer：界面设计

### 3.3 模型推理模块

### 3.4 结果可视化模块

### 3.5 用户交互界面

## 4. 数据流向

```
输入数据 → 预处理 → 模型推理 → 后处理 → 结果展示
   ↑          ↓          ↓          ↓          ↓
   └──────────┴──────────┴──────────┴──────────┘
               数据存储与结果保存
```

## 5. 核心功能流程

### 5.1 图像输入流程
```
用户选择输入方式 → 图像加载 → 预处理 → 模型推理 → 结果展示
```

### 5.2 模型推理流程
```
图像输入 → 模型加载 → 推理计算 → 结果解析 → 可视化展示
```

### 5.3 结果输出流程
```
检测结果 → 数据格式化 → 可视化处理 → 界面展示 → 结果保存
```
